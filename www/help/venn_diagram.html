Help on: Venn diagram

What is a Venn diagram?
A Venn diagram allows us to draw a diagram that will tell us how many genes are differentially expressed in each class comparison. This is done by drawing a circle that represents a class comparison (class 1 vs class 2) and writing in the inside the number of genes that are differentially expressed and outside the circle the number that are not.

What does the number in the intersection of the circles mean?
Since each circle represents a class comparison (class 1 vs class 2 and class 2 vs class 3), the number in the intersection of the circles represents genes that are at the same time differentially expressed in both comparisons (class 1 differentially expressed with respect to class 2 and class 2 differentially expressed with respect to class 3).

What do each of the three diagrams represent?
Each one of the three diagrams represent the cases where the genes inside the circle are over-expressded, under-expressded or both over-expressded and under-expressded

How do I draw a Venn diagram?
You must select two or more different class comparisons and click on the draw venn diagram button.
