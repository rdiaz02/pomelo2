Help on: Changing FDR

What is the FDR limit for?
A gene with a FDR value below the FDR limit is considered to be differentially expressed. This is used for the differential expression table and to draw Venn diagrams.

What FDR limit is being used?
The FDR limit being used is written in orange. By default the limit is set at 0.05

How can I change it?
You must type the new value in the textbox and click on Change criteria. The new value should appear written in orange. This will also automatically redraw diagrams and recalculate the differential expression table.
