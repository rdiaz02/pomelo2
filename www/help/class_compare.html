Help on: Class compare

How to make a class comparison?
To compare two classes, you must choose the two classes from the list and click on calculate comparison.

What happens after a class comparison?
Once you have made a class comparison a link to a table with FDR, pvalue, etc will appear below "Results Tables:". This comparison will also appear in the differential expression section.

Clear all comparisons
If you wish to erase all comparisons and start comparisons again, click on clear all comparisons.

Is this the same as a limma t-test between the two classes?
Not exactly. Limma methods borrow information from gene expression data across all classes. So even if we just compare two classes, we will also be using information from the other classes. This means that the results from a class comparison should be better than a limma t-test of those two classes.
