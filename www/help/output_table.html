Help on: Output table

What is the output table?
The output table is a table containing the results of the test you have launched.

What do the values that appear mean?
The header indicates: the test used, the number of permutations (if permutation test) and the covariables that were used (only for annova limma, more information in full help). The actual table contains: table row number, Gene Name, row number (corresponds to order used in your file), unadjusted p-value (p-value with no multiple testing correction), FDR-indep (FDR-adjusted p-value), observed statistic (depends on test used, see full help) and absoulte observed statistic.
More information in full help.

Reorder table and download?
If you wish to reorder the table, you will find a drop down list at the bottom of the table with different ordering options. To download text version of the table and other files, you must click on the "Download output files" button at the bottom of the table.

Send to PaLS?
If you wish to send a list of the genes to PaLS, click on the send to PaLS button. This will take you to a window which will allow you to select the genes that will be sent to PaLS.
