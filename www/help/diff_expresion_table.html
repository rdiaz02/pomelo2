Help on: Differential expression table

How does the differential expression table work?
After making a class comparison (Compare classes box), a table should appear showing the genes that are differentially expressed. The genes that are up-regulated will have a green class1 > class2 and the ones that are down-regulated will have a red class1 < class2.

What is the differential expression criteria?
If a gene has a FDR value below the FDR value that appears in orange in the differntial expression criteria section, it is considered to be differentially expressed. The FDR limit can be changed in the differntial expression criteria section.

Why do no genes appear after I have made a class comparison?
The table only shows genes that are differentially expressed in at least one of the class comparisons. If no genes appear, it means that none of the genes are differentially expressed in the class comparisons you have calculated so far.
